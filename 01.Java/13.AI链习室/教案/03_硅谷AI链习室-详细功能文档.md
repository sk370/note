# 硅谷AI链习室功能详细介绍

本文档详细介绍了智慧学习平台所包含的各大功能模块及其具体功能点。

## 一、 系统信息管理

本模块负责管理系统级别的基本信息，主要包括轮播图和公告。

### 1.1 轮播图管理（8套）

- **更新轮播图**: 修改现有轮播图的图片、标题、链接等信息。
- **切换轮播图状态**: 控制轮播图的启用或禁用。
- **上传轮播图图片**: 上传新的轮播图图片。
- **添加轮播图**: 创建新的轮播图。
- **根据ID获取轮播图**: 查询单个轮播图的详细信息。
- **获取所有轮播图**: 获取所有轮播图的列表，供后台管理使用。
- **获取启用的轮播图**: 获取前台展示的已启用轮播图列表。
- **删除轮播图**: 删除指定的轮播图。

### 1.2 公告管理 （8套）【实战】

- **更新公告信息**: 修改现有公告的标题、内容、类型等。
- **切换公告状态**: 控制公告的启用或禁用。
- **发布新公告**: 创建并发布新的系统公告。
- **根据ID获取公告**: 查询单个公告的详细信息。
- **获取所有公告**: 获取所有公告的列表，供后台管理使用。
- **获取最新公告**: 获取最新发布的若干条公告。
- **获取启用的公告**: 获取前台展示的已启用公告列表。
- **删除公告**: 删除指定的公告。

## 二、 试题信息管理 

本模块负责管理考试系统的核心资源——试题，包括试题的分类和内容管理。

### 2.1 分类管理（5套）

- **获取分类列表**: 查询所有试题分类及其下的题目数量。
- **添加新分类**: 创建新的试题分类。
- **更新分类信息**: 修改现有分类的名称、排序等。
- **获取分类树形结构**: 以层级结构展示所有分类。
- **删除分类**: 删除指定的试题分类。

### 2.2 题目管理（6套）

- **根据ID查询题目详情**: 获取单个题目的完整信息。
- **更新题目信息**: 修改题目的内容、选项、答案等。
- **删除题目**: 删除指定的题目。
- **创建新题目**: 手动添加新的试题。
- **获取热门题目**: 查询访问次数最多的热门题目。
- **分页查询题目列表**: 根据多种条件筛选并分页查询题目。

### 2.3 题目批量操作（5套）

- **预览Excel文件内容**: 解析并预览Excel文件中的题目。
- **批量导入题目**: 将题目列表批量存入数据库。
- **从Excel文件批量导入题目**: 直接从Excel文件导入题目。
- **AI智能生成题目**: 根据要求使用AI自动生成题目。
- **下载Excel导入模板**: 提供用于批量导入的Excel模板文件。

## 三、 考试信息管理 

本模块负责考试的全流程管理，包括考试的创建、进行、批阅和记录查询。

### 3.1 考试管理（4套）

- **提交考试答案**: 学生提交答卷。
- **AI自动批阅**: 使用AI技术自动为考试评分。
- **开始考试**: 学生开始一场新的考试。
- **查询考试记录详情**: 查看某次考试的详细记录。

### 3.2 试卷管理（7套）

- **获取试卷详情**: 查看试卷的详细信息和包含的题目。
- **更新试卷信息**: 修改试卷的名称、题目配置等。
- **删除试卷**: 删除指定的试卷。
- **手动创建试卷**: 通过手动选题的方式创建试卷。
- **更新试卷状态**: 发布或停止试卷。
- **AI智能组卷**: 根据规则自动生成试卷。
- **获取试卷列表**: 查询所有试卷的列表。

### 3.3 考试记录管理（3套）

- **获取考试记录列表**: 查看指定考试记录的详细信息。
- **删除考试记录详情**：删除指定考试记录详情
- **考试排行记录接口**：查询考试排行信息进行展示

## 四、 视频信息管理【实战】

本模块负责管理平台上的教学视频资源。

### 4.1 视频分类管理

- **获取视频分类列表**: 查询所有视频分类。
- **添加新分类**: 创建新的视频分类。
- **更新分类信息**: 修改现有分类的名称、排序等。
- **获取分类树形结构**: 以层级结构展示所有视频分类。
- **删除分类**: 删除指定的视频分类。

### 4.2 视频管理

- **根据ID查询视频详情**: 获取单个视频的详细信息。
- **更新视频信息**: 修改视频的标题、描述、URL等。
- **删除视频**: 删除指定的视频。
- **创建新视频**: 添加新的视频资源。
- **获取热门视频**: 查询播放次数最多的热门视频。
- **分页查询视频列表**: 根据条件筛选并分页查询视频。
